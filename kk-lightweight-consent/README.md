# KK Lightweight Consent

WordPress plugin for a lightweight cookie consent banner with Google Tag Manager and Google Consent Mode v2 support.

## Installation

1. Zip the `kk-lightweight-consent` folder.
2. In WordPress admin, go to **Plugins → Add New → Upload Plugin**.
3. Upload the ZIP and activate **KK Lightweight Consent**.

## Included files

- `kk-lightweight-consent.php` (main plugin file)
- `assets/kk-consent.js`
- `assets/kk-consent.css`
